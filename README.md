# noetix_sdk_bumi
Noetix robot Bumi sdk.

### Prebuild environment
* OS  (Ubuntu 20.04 LTS & Ubuntu 22.04 LTS)  
* CPU  (aarch64 or x86_64)   
* Compiler  (gcc version 9.4.0) 

### Environment Setup

Before building or running the SDK, you can read the documentation linked here to ensure the following dependencies are installed:

[Environment Setup](https://noetixrobotics.feishu.cn/docx/PP2BdXkSUoQ2VOxctJKcEdHanAg?from=from_copylink).

### Build examples

To build the examples inside this repository:

```bash
./build_release.sh
```

For more reference information, please go to [Bumi DDS_SDK](https://noetixrobotics.feishu.cn/docx/U4radEDEpoUznExqrqBcgrKinhd).
[Bumi DDS_SDK PYTHON](https://noetixrobotics.feishu.cn/wiki/NoUOwvFBFiXmVVkm7NSczqbVnKd).
